Detailed Description

Parser class that accepts only characters given.

Refer to the usage below.

Utility::CharGroupParser cgp("ab");
cgp.parse("a"); // == true
cgp.index(); // == 0
cgp.parse("b"); // == true
cgp.index(); // == 1
cgp.parse("ax"); // == true
cgp.index(); // == 0
cgp.parse("c"); // == false
cgp.index(); // == -1
