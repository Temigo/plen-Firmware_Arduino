Detailed Description

与えられた文字グループのみ受理するパーサクラス

Note
CharGroupPurserは内部でリニアサーチを行うため、 受理優先度の高い文字ほど先頭に配置すると効果的です。

パースを行うメソッド

Note
現在の実装では、与えられた文字列の先頭に対してのみパースを行います。 すなわち、/^[accept_chars]/であるかを検証します。

/^[accept_chars]+$/であるかを検証する挙動に変えたい場合は、 直接ソースコードを編集してください。
